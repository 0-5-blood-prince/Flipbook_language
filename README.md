# Flipbook_language

fc.py is the compiler /n
Default : It creates an pdf of the defined flipbook
To Create Pdfs
```
./fc.py life.flip
```
```
./fc.py life.flip -pdf
```
```
./fc.py life.flip -pdf images.pdf
```
To Creates Gifs
```
./fc.py life.flip -gif
```
```
./fc.py life.flip -gif lifecycle.gif
```
